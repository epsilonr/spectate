# [ZR] Spec Players
### Commands

* sm_afk
* sm_spec <name|#userid>
* sm_spectate <name|#userid>
* sm_observe <name|#userid>

### Cvars

* sm_spec_enabled "1/0" Enable / Disable plugin. (def: 1)
* sm_spec_blockmotherzm "1/0" Blocks motherzombies to use !spec command. 1 = Block | 0 = Don't block. (def: 1)
